# 📺 YouTube Clone

A front-end clone of YouTube's homepage built with pure HTML and CSS — no frameworks, no JavaScript libraries.

## 🔍 Preview

A pixel-accurate recreation of YouTube's home feed, including:
- Fixed header with search bar, mic button, create button, notifications, and profile
- Collapsible-style sidebar with Home, Shorts, Subscriptions, and You
- Responsive video grid with thumbnails, channel info, view counts, and video durations

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox & Grid)
- Google Fonts (Roboto)

## 📁 Project Structure

```
youtube-clone/
├── index.html
├── General.css
├── Header.css
├── Sidebar.css
├── video.css
├── Header/          # Header icons and logo
├── sidebar/         # Sidebar icons
├── Thumbnails/      # Video thumbnails
└── Profile-pic/     # Channel profile pictures
```

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Arockia-Joyston/youtube-clone.git
   ```
2. Open `index.html` in your browser — no setup needed.

## ✨ Features

- Fixed top navigation bar
- Functional search bar UI
- Tooltip hover effects on header buttons
- Notification badge on the bell icon
- 3-column video grid layout
- Video duration overlays on thumbnails
- Clickable video and channel links
